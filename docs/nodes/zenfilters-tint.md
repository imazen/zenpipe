# ⚙️ Tint

> **ID:** `zenfilters.tint` · **Role:** filter · **Group:** color

Green-magenta tint adjustment via Oklab a shift.  Positive values shift toward magenta, negative toward green.

## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `shift` | float (-1.0 – 1.0) | 0.0 | Tint shift (negative = green, positive = magenta) |
